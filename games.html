Welcome to an immersive island survival experience! Stranded on a remote, mysterious island, players face the test of endurance and adaptability. Key features include:

  • Dynamic Day-Night Cycle: A basic day night cycle.
  • Resource Gathering and Base Building: Collect materials to craft tools, build shelters, and establish your base.
  • Combat Mechanics: Introduce basic combat mechanics
  • Survival Systems: A basic Stats system that asks the player to take look over Health, Hunger, Thirst and Stamina
  • Enemy AI: A basic form of AI which uses the Player position for pathfinding

This game is a basic survival template, blending exploration, strategy, and action in a vibrant island.

A Souls-like RPG game where I used a mixture of C++ and Blueprints to create combat mechanics, enemy AI. The game features dynamic animations, stamina-based combat, and a variety of unique move sets.

  • Combat System: Features blocking mechanics and different attack animations.
  • Enemy AI: Implemented pathfinding, attack patterns, and a decision tree for the AI to switch between Ranged, Melee, and Charge states, each with its unique animation.
  • Level Design: Created a basic test world where the player can try out core gameplay mechanics such as blocking, attacking, and running.
  • Lock On System: Implemented a Lock-on system for the player to be able to lock onto the enemy to have an easier time fighting.
